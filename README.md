# AAA Kitchenware Classification – Group 9

## Overview
This repository contains the work developed for the Group 9 practical study on image classification of kitchenware objects.  
The project explores multiple deep learning architectures and compares their performance across different experimental setups.

## Project Structure

- `notebooks/`: All experiments (baseline CNN, ResNet50, EfficientNet, ViT, etc.)
- `docs/`: Report, presentation materials, and assignment guidelines
- `results/`: Confusion matrices and training curves
- `data/`: Dataset instructions (not stored due to size)

---

## Datasets

Two datasets were used:

1. **Kaggle Kitchenware Dataset**  
   https://www.kaggle.com/datasets/agrigorev/kitchenware-classification  

2. **EVEN/ODD dataset (ULE Agora platform)**  
   Access provided via university platform.

All dataset download and setup instructions are available in `data/README.md`.

---

## Experiments

The project includes multiple experiments:

- Baseline CNN
- ResNet50 transfer learning
- EfficientNet variants
- Vision Transformer (ViT)

Each experiment is implemented in a separate notebook inside `notebooks/`.
